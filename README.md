#### React 的核心概念
1.JSX的核心语法（javascript和XML 结合的语法）
2.元素渲染（页面渲染）
3.组件 创建组件 和 组件之间传参
4.props 属性 是父元素（父组件传递给子组件的值）  和  state 状态（子组件自己创建的值）
5.事件处理 添加事件 以及在事件里面获取this
6.条件循环 根据条件来渲染某个组件
7.列表循环
